# Employee Attrition Analysis and Workforce Insights Dashboard

## Project Overview

This project analyzes employee attrition patterns using SQL, Excel, and Power BI. The objective is to identify factors contributing to employee turnover and provide business insights that can support workforce planning and retention strategies.

## Tools and Technologies

* MySQL
* Microsoft Excel
* Power BI
* SQL
* Power Query
* DAX

## Dataset Information

Dataset: IBM HR Analytics Employee Attrition Dataset

Records: 1,470 Employees

Attributes: 36 Employee-related Features

## Project Workflow

1. Data Collection
2. Data Cleaning and Validation
3. SQL Analysis
4. Power BI Dashboard Development
5. Business Insight Generation
6. Recommendations and Conclusions

## Dashboard KPIs

* Total Employees: 1,470
* Attrition Count: 237
* Attrition Rate: 16.12%

## Key Insights

### Department Analysis

* Sales department recorded the highest attrition rate.
* Research & Development contributed the highest number of attrition cases.

### Gender Analysis

* Male employees contributed more to employee attrition than female employees.

### Overtime Analysis

* Employees working overtime demonstrated higher employee turnover.

### Job Role Analysis

Highest attrition was observed among:

* Laboratory Technicians
* Sales Executives
* Research Scientists

## Business Recommendations

* Improve employee retention programs in high-attrition departments.
* Monitor employee workload and overtime.
* Conduct employee satisfaction surveys for high-risk job roles.
* Enhance career growth and employee engagement initiatives.
